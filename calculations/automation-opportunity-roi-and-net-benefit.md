# Automation Opportunity ROI and Net Benefit

**Purpose:** Present total benefit, net benefit, and ROI for each automation opportunity using the same structure and assumptions as the business-case document.

**Definitions:**
- Total benefit = Annual cost saved + Annual revenue uplift
- Net benefit = Total benefit - Implementation cost
- ROI % = (Net benefit / Implementation cost) x 100

**Source assumptions used:**
- OP-01: Cost saved = 5,793; Revenue uplift = 480,000; Implementation cost = 45,000
- OP-02: Cost saved = 13,268; Revenue uplift = 1,440,000; Implementation cost = 102,000
- OP-03: Cost saved = 5,844; Revenue uplift = 1,920,000; Implementation cost = 45,000
- OP-04: Cost saved = 1,562; Revenue uplift = 960,000; Implementation cost = 85,000
- OP-05: Cost saved = 27,588; Revenue uplift = 2,400,000; Implementation cost = 85,000

---

## Individual Opportunity Summary

### OP-01: Follow-up management

| Metric | Value |
|--------|-------|
| **Implementation cost** | 45,000 |
| **Annual cost saved** | 5,793 |
| **Annual revenue uplift** | 480,000 |
| **Total benefit** | 485,793 |
| **Net benefit** | 440,793 |
| **ROI %** | 979.54% |

**Calculation:**
```
Total benefit = 5,793 + 480,000 = 485,793
Net benefit = 485,793 - 45,000 = 440,793
ROI % = (440,793 / 45,000) x 100 = 979.54%
```
